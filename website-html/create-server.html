Create a Discord Server

Want to create a Discord server for a UCSD course?

  • Make your server!

    Head to discord.com
    Either Login in on the top right or select “Open Discord in browser”.

    Simply clicking here will redirect you to a template designed for UCSD classes.

    Fill in the name accordingly and then add your server to our site by scrolling below!

Customize your server with bots!

Consider adding our custom Discord bot to your server! Simply click this link, make sure you are an admin, and select add to server!

  • Event Commands

    Keep track of events and
    be notified on the day they happen!
    !setevent event name
    !deleteevent event name
    !schedule

  • Assignment Commands

    Keep track of assignments and
    be reminded on the due date!
    !addAssign assignment name mm/dd/yyyy
    !delAssign assignment name
    !listAssign

  • Rules Commands

    Make rules for members to follow!
    !deleterule number
    !addrule "title" description
    !rule number optional, lists all rules if no argument
    !swaprules num1 num2

  • Report Commands

    React to a message with the red square reaction
    to report a message to the admins of the server
    in the case of discrimination or an academic
    integrity violation.

Now Add it to Our Website!

Important Reminder
Ensure that the Discord invite link you share is a permanent invite. To create a permanent invite, select "never" under "expire after" and "no limit" under "max uses." Don't know to retrieve the invite link? Look no further!
