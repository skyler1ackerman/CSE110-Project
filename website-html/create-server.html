Create a Discord Server

Want to create a Discord server for a UCSD course?

  • Make your server!

    1. Head to discord.com to login.
    Either Log into the app from top right or click “Open Discord in browser."

    2. Simply clicking here will redirect you to a template designed for UCSD classes.

    3. Fill in the server name accordingly and then add your server to our site so that others can find you!

    • Customize your server with bots!

    Consider adding our custom Discord bot to your server! Simply click this link, make sure you are an admin, and select add to server!

  •  Event Commands

    Keep track of events and be notified on the day they happen!
    Command Format:

    •   !addEvent <EventName>
    •   !deleteEvent <EventName>
    •   !listEvent <EventName>

  •  Assignment Commands

    Keep track of assignments and be reminded on the due date!
    Command Format:

    •   !listAssign
    •   !deleteAssign <AssignName>
    •   !addAssign <AssignName> mm/dd/yyyy

  •  Rules Commands

    Make rules for members to follow!
    Command Format:

    •   !deleteRule <RuleName>
    •   !addRule "title" <RuleDesc>
    •   !swapRule <RuleNum1> <RuleNum2>
    •   !listRule <RuleNum> (optional)

  •  Report Commands

    In case of server policy or academic integrity violation: React to a message with the red square reaction to report a message to the admins of the server.
